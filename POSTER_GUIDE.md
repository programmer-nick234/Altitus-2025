## Image Specifications

- **Aspect Ratio:** 16:9 (Landscape)
- **Recommended Size:** 1920x1080px or 1280x720px
- **Format:** JPG, PNG, or WebP
- **Max File Size:** 500KB (for performance)

## Placeholder Behavior

If a poster image is missing, the system will show:
- Generic placeholder with department color
- Event name initial (e.g., "C" for Code Sprint)
- Department color as background

**Note:** Until you add real posters, the system will use colored placeholders automatically.
